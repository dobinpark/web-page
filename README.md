# 심플몰(Simple Mall)

# 💡 Topic
* **쇼핑몰 운영에 필요한 핵심적인 로직이 담긴 플랫폼**
* 첫 클론 코딩이자 개발자의 임의로 추가적인 기능들을 구현

# 📝 Summary
초기 스타트업이 아무것도 없이 조금씩 성장하면서 체계적인 프로세스가 생기듯이,
백엔드 개발자로서 처음 프로젝트를 시작할 때 필요한 지식과 기술들을 습득하면서 앞으로
진행되는 프로젝트를 진행할 때 필요한 초석이라 생각하고 제작하게 되었습니다.

# ⭐️ Key Function
* 회원 기능
  * 회원가입을 할 때, 주로 입력하는 이름과 이메일, 주소와 상세 주소를 입력하고 데이터가 저장이 되면 회원 목록에 가입한 회원 리스트를 확인할 수 있음
  
* 상품 기능
  * 상품 등록을 할 때, 상품명, 가격, 수량을 입력하고 데이터가 저장이 되면 상품 목록에 저장된 상품 리스트를 확인할 수 있음
  
* 주문 기능
  * 상품 주문을 할 때, 가입된 회원과 저장된 상품을 기입하고 추가로 기능 구현한 색상과 사이즈를 입력하고 데이터가 저장이 되면 주문 내역에 주문한 상품 리스트를 확인할 수 있음
  * 리스트에서 주문상태를 확인할 수 있는데 주문 취소를 하고 싶을 시, 주문 취소 버튼을 누르고 나서 주문 상태(ORDER, CANCEL)를 검색하면 구별된 상품을 조회할 수 있음

# ⚙️ Architecture
MVC

**전체 개발**

# 🤔 Learned
* Java/Spring의 개념과 동작 원리를 이해하고, 직접 코드를 작성하면서 적용하는 방법을 알게 되었음. 또한, 백엔드 개발자가 가져야 할 기술적인 역량을 확인할 수 있었음.
* 웹 시스템 구성인 Client ←→ Web ←→ Was ←→ DB의 동작 원리를 알게 되었음.
* 프로젝트를 진행할 때, JPA 활용의 중요성을 알게 됨.
* JPA를 이용하여 DB(MySQL)를 활용할 수 있었음.
* 클론 프로젝트였지만 기능 구현의 욕심이 생겨 다음 프로젝트때 웹 설계의 중요성을 가지게 되어 Spring과 JPA, DB에 대해 공부할 수 있었음.
